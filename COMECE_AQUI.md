## ⚡ 3 Passos Rápidos

### 1. Testar com Dados de Exemplo (2 min)
```
1. Abra http://localhost:3000/demo.html
2. Clique "📰 Carregar Dados de Exemplo"
3. Clique "👁️ Abrir Jornal Público"
→ Verá 4 artigos em formato jornal elegante
```

### 2. Usar com Suas Notas (5 min)
```
1. Abra http://localhost:3000/index.html
2. Crie uma nota (título + conteúdo)
3. Clique "Salvar"
4. Clique "📰 Compartilhar"
5. Clique "Compartilhar Agora"
6. Copie o link com "Copiar Link"
```

### 3. Compartilhar (sempre)
```
1. Envie o link public.html para amigos
2. Eles verão todas suas notas no formato jornal
3. Sem necessidade de login ou configuração!
```

## 🐛 Problemas?

**"Botão não aparece"**
→ Limpe cache do navegador (Ctrl+Shift+Del)

**"Dados não salvam"**
→ Verifique se localStorage está ativado na privacidade

**"Página pública vazia"**
→ Compartilhe uma nota pelo app primeiro!

**"Design errado"**
→ Verifique se Google Fonts carregou em devtools (F12)

Para mais: Veja **GUIA_RAPIDO.txt**
