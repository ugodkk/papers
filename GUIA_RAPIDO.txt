💾 DADOS E ARMAZENAMENTO
====================================================================

ONDE OS DADOS SÃO SALVOS?
Local Storage do Navegador
Chave: "shared_notes"
Formato: JSON Array

EXEMPLO DE ESTRUTURA:
{
  "id": 1707414600000,
  "title": "Meu Artigo",
  "content": "Conteúdo completo aqui...",
  "tags": ["Categoria1", "Categoria2"],
  "date": "27/02/2026",
  "sharedAt": "2026-02-27T15:30:00Z"
}

LIMITES:
- ~5-10MB por navegador (suficiente para centenas de notas)
- Persiste mesmo após fechar navegador
- Precisa fazer backup manualmente

BACKUP:
Use a função no newspaper.js:
  newspaper.exportAsJSON()  // Salva como arquivo JSON

🛠️ TROUBLESHOOTING
====================================================================

PROBLEMA: Botão "Compartilhar" não aparece
SOLUÇÃO: Certifique-se que newspaper.js foi carregado
         Verifique console (F12) para erros

PROBLEMA: Modal não abre
SOLUÇÃO: Verifique se script.js carregou corretamente
         Veja console para erros de JavaScript

PROBLEMA: Dados não salvam
SOLUÇÃO: localStorage pode estar desabilidato
         Verifique privacidade do navegador
         Tente em navegador diferente

PROBLEMA: Página pública não mostra notas
SOLUÇÃO: Nenhuma nota foi compartilhada ainda
         Use demo.html para carregar exemplos
         Ou crie e compartilhe uma nota em index.html

PROBLEMA: Design fica errado
SOLUÇÃO: Limpe cache do navegador (Ctrl+Shift+Del)
         Verifique se CSS carregou (fonte)
         Teste em navegador diferente

📞 PERGUNTAS FREQUENTES
====================================================================

P: É preciso fazer login?
R: Não! Tudo funciona sem autenticação

P: Os dados são sincronizados entre dispositivos?
R: Não, são locais. Mas você pode exportar/importar

P: Posso editar depois de compartilhar?
R: Sim! Compartilhe novamente com alterações

P: Como deletar uma nota compartilhada?
R: Use newspaper.unshareNote() ou limpe em demo.html

P: Qual é o limite de caracteres?
R: ~5-10MB por navegador (muito espaço!)

P: Posso ter múltiplos jornais?
R: Não no mesmo localStorage, mas diferentes é possível com customização

P: Funciona offline?
R: Sim! Tudo funciona sem internet

P: Como fazer backup?
R: Use newspaper.exportAsJSON() para salvar

P: Posso customizar as cores?
R: Sim! As cores estão em /public.html com comentários

P: Preciso de hosting?
R: Qualquer servidor web funciona. Também funciona local!
