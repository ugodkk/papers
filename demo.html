🎯 Como Usar Este Demo?

Este página permite você pré-visualizar como o jornal funcionará com dados de exemplo.

  1. Carregar Exemplos: Clique em "Carregar Dados de Exemplo" para popular o jornal
  2. Ver Jornal: Clique em "Abrir Jornal Público" para ver como fica
  3. Limpar Dados: Use "Limpar Dados" para remover os exemplos
  4. Adicionar Manual: Acesse o app principal (index.html) para adicionar suas próprias notas
