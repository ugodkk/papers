-----------
- Título: 4em em Playfair Display (900 weight)

- Linhas 1.7-1.8 para melhor legibilidade

🚀 COMO USAR - GUIA RÁPIDO
====================================================================

PASSO 1: Abrir o Demonstrativo
--------------------------------
1. Abra "demo.html"
2. Clique em "📰 Carregar Dados de Exemplo"
3. Clique em "👁️ Abrir Jornal Público"
4. Veja como fica o jornal com dados!

PASSO 2: Usar com Suas Notas
------------------------------
1. Abra "index.html"
2. Crie uma nota normal
3. Clique em "📰 Compartilhar"
4. Modal abrirá mostrando preview
5. Clique em "Compartilhar Agora"
6. Copie o link com "Copiar Link"
7. Compartilhe em qualquer lugar!

PASSO 3: Ver o Jornal Público
------------------------------
1. Abra "public.html" diretamente
2. Ou clique no link compartilhado
3. Verá todas as notas em formato jornal
4. Layout elegante e profissional

💾 ARMAZENAMENTO DE DADOS
====================================================================

LOCAL STORAGE:
--------------
Chave: "shared_notes"
Tipo: JSON Array

Estrutura:
{
  "id": 1234567890,           // Timestamp único
  "title": "Título da Nota",   // Título da nota
  "content": "Conteúdo...",    // Texto completo
  "tags": ["Tag1", "Tag2"],    // Categorias
  "date": "27/02/2026",        // Data de publicação
  "sharedAt": "ISO-String"     // Timestamp de compartilhamento
}

CAPACIDADE:
-----------
- LocalStorage: ~5-10MB por site (navegador)
- Suficiente para centenas de notas
- Persiste mesmo após fechar o navegador
- Sincroniza automaticamente
